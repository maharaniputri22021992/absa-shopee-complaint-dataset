# ABSA Shopee Complaint Dataset

Dataset ini berisi 1799 ulasan keluhan pelanggan Shopee dari Google Play Store yang telah dianotasi untuk penelitian Aspect-Based Sentiment Analysis.

## Kolom Dataset

- ulasan : teks keluhan pelanggan
- delivery : label aspek pengiriman
- payment : label aspek pembayaran
- cs_attitude : label sikap customer service
- app_performance : label performa aplikasi

## Format Label
1 = aspek muncul
0 = aspek tidak muncul

Dataset digunakan dalam penelitian skripsi mengenai Aspect-Based Sentiment Analysis pada ulasan e-commerce Indonesia.
